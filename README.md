# 🦢 블랙스완 주식 시뮬레이터

나심 탈레브의 사상에 기반한 주식 시장 시뮬레이터. **정규분포(가우시안)가 아니라 멱법칙
(팻테일) + 블랙스완**을 전제로, 두 가지 도구를 한 화면에서 제공한다.

## 두 가지 모드

### 📈 가격 확률 예측 (기본)

아이폰에서 **종목을 검색·선택 → 시행 횟수·기간 설정 → 가격의 확률적 인사이트**를 얻는다.

- **자동 보정:** 선택한 종목의 과거 일봉으로 현재가·연변동성·드리프트·**멱법칙 꼬리지수
  α(Hill 추정)**를 계산해 시뮬에 주입. 직접 조절도 가능.
- **인사이트:** 만기 가격 분위수(P1~P99) · 기대 가격 · **목표가 도달 확률** · 꼬리 위험
  (VaR / 기대손실 CVaR) · 손실 임계 확률(−10/20/35/50%) · 최대낙폭 분포.
- **시각화:** 시간대별 가격 팬차트(P5–95 / P25–75 구간) · 만기 수익률 히스토그램에
  **정규분포 베이스라인 오버레이** — 가우시안이 극단을 얼마나 과소평가하는지 한눈에.

### 🛡 전략 프래질리티

**핵심은 예측이 맞느냐가 아니라, 예측에 근거한 행동(전략)이 블랙스완 앞에서 프래질(취약)
한가, 안티프래질(반취약)한가**를 평가하는 것.

## 무엇을 평가하나 (프래질리티 모드)

- **볼록성 지수 (안티프래질리티):** 변동성을 ±50% 섭동시켜 손익의 2차 곡률을 측정하는
  탈레브–두아디(Taleb–Douady) 휴리스틱. `>0` 안티프래질 · `<0` 프래질 · `≈0` 로버스트.
- **파산 확률 (생존 게이트):** 자본의 절반 이상을 잃을 확률. 탈레브의 흡수장벽/비에르고딕성
  원칙 — 기대수익이 높아도 파산확률이 0이 아니면 그 행동은 피한다. 의사결정의 1순위.
- **꼬리 비대칭 / 최악 경로 손실 / 손익 분포(팻테일).**

## 작동 방식

- **시뮬레이션 코어**: GBM에 **멱법칙 점프(꼬리지수 α)** + 변동성 클러스터링 + 블랙스완
  충격을 얹은 점프확산(Bates) 모델. 두 모드가 같은 엔진(`paths.ts`)을 공유한다.
- **데이터 레이어**: [Twelve Data](https://twelvedata.com) API를 Next.js route handler로
  프록시(`/api/search`, `/api/quote`). 서버리스 함수가 시세·과거데이터만 중계하고, 무거운
  몬테카를로는 여전히 **브라우저 Web Worker**에서 돈다. 아이폰에서 매끄럽게.
  - **환경변수**: `TWELVE_DATA_API_KEY`(무료 발급)를 Vercel 프로젝트 env에 설정.
    데이터센터 IP를 차단하는 무키 소스(Yahoo 등)의 429 문제를 피하기 위함.
  - 키가 없거나 제공자가 응답하지 않으면 UI가 **수동 입력 폴백**으로 전환된다
    (티커·현재가·연 변동성 직접 입력 → 동일 시뮬 실행).
- 프래질리티 모드 프리셋 전략: 바이앤홀드 · 바벨(90% 현금+10% OTM 콜) · 테일 헤지 · 숏 볼
  · 네이키드 풋 매도 · 커버드 콜 · 전액 현금.

## 스택 / 배포

- Next.js 14 (App Router) + TypeScript + React + recharts.
- Vercel 배포(제로컨피그, 키 불필요). 엔진·보정 정합성은 vitest로 검증.

## 개발

```bash
npm install
npm run dev      # http://localhost:3000
npm test         # 보정(calibrate) · 예측(forecast) · 프래질리티 엔진 정합성 테스트
npm run build
```

## 코드 구조

- `lib/sim/` — 시뮬레이션 엔진 (순수 TS, UI 비의존)
  - `paths.ts` 멱법칙 점프확산 경로 + 블랙스완 주입(꼬리지수 α·팬차트 기록 옵션) ·
    `forecast.ts` 가격 분포·꼬리 위험·팬차트·가우시안 비교 진입점 ·
    `payoff.ts` 전략/레그 손익 · `fragility.ts` 탈레브–두아디 휴리스틱 ·
    `blackScholes.ts` 옵션 가격책정 · `engine.ts` 프래질리티 진입점 · `worker.ts` Web Worker.
- `lib/calibrate.ts` — 과거 종가 → 드리프트·변동성·Hill 꼬리지수 자동 보정.
- `app/api/` — `search`·`quote` route handler (Twelve Data 시세 프록시, `TWELVE_DATA_API_KEY`).
- `components/` — `StockPicker`·`ForecastControls`·`PriceInsights`·`PriceFanChart`·
  `PriceHistogram`(가격 모드) + 프래질리티 대시보드 UI · `app/` — Next.js 페이지.
- `reference/` — 원본 Streamlit 프로토타입(`app.py`). 수식 포팅의 참고 자료.

> ⚠️ 교육용 시뮬레이터입니다. 과거·합성 데이터 기반이며 투자 자문이 아닙니다.
